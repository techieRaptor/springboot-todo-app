# Spring Boot To-Do App 📝
A simple To-Do application built with Spring Boot and H2 database.

## Features:
✅ Add, update, delete tasks  
✅ Store tasks in H2 database  
✅ REST API endpoints  

## How to Run:
1. Clone this repo  
2. Run `mvn spring-boot:run`  
3. Open `http://localhost:8080/h2-console`
